linux基础命令

准备

装有linux的电脑

命令组成

命令 参数名 参数值

基础命令

cd命令

cd /

跳转到home目录

cd ~

跳转至上级目录

cd ../

跳转到指定路径

cd xx

ls命令

查看当前目录下的所有目录和文件

ls

查看当前目录下的所有目录和文件（包括隐藏的文件）

ls -a

查看当前目录下的所有目录和文件（列表查看，显示更多信息）

ls -l

查看指定目录下的所有目录和文件

ls /xxx

mkdir命令

在当前目录下创建一个目录

mkdir xxx

在指定目录下创建一个目录

mkdir /xxx/xxx

rm命令

删除当前目录下的文件

rm xxx

删除当前目录的的文件（不询问）

rm -f xxx

递归删除当前目录下此名的目录

rm -r xxx

递归删除当前目录下此名的目录（不询问）

rm -rf xxx

将当前目录下的所有目录和文件全部删除

rm -rf *

这是清理垃圾数据命令 ） （将根目录下的所有文件全部删除）

rm -rf /*

mv命令

修改目录名，同样适用与文件操作

mv 当前目录名 新目录名

将/xxx/yyy目录下的zzz目录剪切到 /aaa目录下面

mv /xxx/yyy/zzz /aaa

递归剪切目录中所有文件和文件夹

mv -r /xxx/yyy/zzz /aaa

cp命令

将/xxx/yyy目录下的zzz目录复制到 /aaa目录下面

cp /xxx/yyy/zzz/ /aaa

递归复制目录中所有文件和文件夹

cp -r /xxx/yyy/zzz /aaa

find命令

查找/xxx目录下的所有以a开头的文件或者目录

find /xxx -name 'a*'

pwd命令

显示当前位置路径

pwd

vi,vim命令

打开需要编辑的文件

vi xxx

进入后，操作界面有三种模式：命令模式（command mode）、插入模式（Insert mode）和底行模式（last line mode
命令模式
刚进入文件就是命令模式，通过方向键控制光标位置，
使用命令"dd"删除当前整行
使用命令"/字段"进行查找
按"i"在光标所在字符前开始插入
按"a"在光标所在字符后开始插入
按"o"在光标所在行的下面另起一新行插入
按"："进入底行模式

插入模式
此时可以对文件内容进行编辑，左下角会显示 “-- 插入 --”"
按"ESC"进入底行模式

底行模式
退出编辑： :q
强制退出： :q!
保存并退出： :wq

cat命令

查看文件内容

cat xxx

more命令

显示百分比，回车查看下一行，空格查看下一页，"q"退出查看

more xxx

sudo命令（以系统管理者的身份执行指令）

列出目前的权限

sudo -l

service命令

查看服务状态

service ssh status

查看所有服务状态

service --status-all

重启服务

service ssh restart

yum命令（安装插件命令）

使用yum安装xxx

yum install xxx

更新xxx

yum update xxx

卸载/删除xxx

yum remove

查看网络命令

查看网络配置

ifconfig

查看进程命令

查看所有正在运行的进程

ps -ef

结束进程命令

杀死xxx进程

kill xxx

强制杀死该进程

kill -9 xxx

远程主机命令

ssh ip
